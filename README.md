# Thiru Tech Institute Website

This is a simple static website for Thiru Tech Institute for IT Course.

## Deployment Instructions

1. Create a new repository on GitHub (go to github.com, sign in, click "New repository").
2. Name it something like "thiru-tech-institute" (or your choice).
3. Do not initialize with README, .gitignore, or license.
4. Copy the repository URL (e.g., https://github.com/yourusername/thiru-tech-institute.git).

Then, in your terminal (in the TTT_TTT folder):

```
git remote add origin <your-repo-url>
git branch -M main
git push -u origin main
```

5. Go to your repository on GitHub, click "Settings" > "Pages".
6. Under "Source", select "Deploy from a branch", choose "main" branch, and save.
7. Your site will be live at https://yourusername.github.io/repositoryname (it may take a few minutes).

## Local Development

Open `index.html` in a web browser to view locally.
