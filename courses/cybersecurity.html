Cybersecurity MCQs

Test your knowledge with these multiple-choice questions on Cybersecurity.

1. What is phishing?

a) A type of fish

b) A cyber attack to steal sensitive information

c) A network protocol

Answer: b) A cyber attack to steal sensitive information

2. Which tool is used for password cracking?

a) Wireshark

b) John the Ripper

c) Nmap

Answer: b) John the Ripper

3. What does VPN stand for?

a) Virtual Private Network

b) Very Private Network

c) Virtual Public Network

Answer: a) Virtual Private Network

4. What is malware?

a) A type of software

b) Malicious software designed to harm

c) A hardware component

Answer: b) Malicious software designed to harm

5. Which is a common encryption standard?

a) AES

b) HTTP

c) FTP

Answer: a) AES
