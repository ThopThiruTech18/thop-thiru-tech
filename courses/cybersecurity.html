Test your knowledge with these Cybersecurity MCQs.


MCQs


1. What is phishing?

a) A type of fish

b) A cyber attack to steal sensitive information

c) A network protocol

Answer: b) A cyber attack to steal sensitive information

2. Which tool is used for password cracking?

a) Wireshark

b) John the Ripper

c) Nmap

Answer: b) John the Ripper

3. What does VPN stand for?

a) Virtual Private Network

b) Very Private Network

c) Virtual Public Network

Answer: a) Virtual Private Network

4. What is malware?

a) A type of software

b) Malicious software designed to harm

c) A hardware component

Answer: b) Malicious software designed to harm

5. Which is a common encryption standard?

a) AES

b) HTTP

c) FTP

Answer: a) AES

Interview Questions


1. What is the CIA triad?

Answer: Confidentiality, Integrity, Availability.

2. Symmetric vs Asymmetric encryption?

Answer: Same key vs public/private keys.

3. What is a zero-day vulnerability?

Answer: Exploited before a patch exists.

4. What is penetration testing?

Answer: Simulated attacks to find weaknesses.

5. IDS vs IPS?

Answer: IDS detects, IPS prevents attacks.
