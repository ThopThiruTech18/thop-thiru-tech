Test your knowledge with these multiple-choice questions on Data Science.

1. What is the primary goal of data science?

a) To store data

b) To extract insights from data

c) To delete data

Answer: b) To extract insights from data

2. Which library is commonly used for data manipulation in Python?

a) NumPy

b) Pandas

c) Matplotlib

Answer: b) Pandas

3. What does SQL stand for?

a) Structured Query Language

b) Simple Query Language

c) Standard Query Language

Answer: a) Structured Query Language

4. Which technique is used for data visualization?

a) Regression

b) Clustering

c) Plotting

Answer: c) Plotting

5. What is a common machine learning algorithm for classification?

a) Linear Regression

b) Decision Tree

c) K-Means

Answer: b) Decision Tree

6. What is data wrangling?

a) Cleaning and preparing data

b) Storing data

c) Deleting data

Answer: a) Cleaning and preparing data

7. Which tool is used for big data processing?

a) Hadoop

b) Excel

c) Word

Answer: a) Hadoop

8. What is a data frame in R or Pandas?

a) A type of chart

b) A 2D data structure

c) A database

Answer: b) A 2D data structure

9. What does ETL stand for?

a) Extract, Transform, Load

b) Edit, Test, Load

c) Enter, Transfer, Leave

Answer: a) Extract, Transform, Load

10. Which is a supervised learning algorithm?

a) PCA

b) SVM

c) K-Means

Answer: b) SVM

Interview Questions

1. What is the difference between supervised and unsupervised learning?

Answer: Supervised uses labeled data, unsupervised does not.

2. Explain overfitting and underfitting.

Answer: Overfitting fits training data too well, underfitting too poorly.

3. What is cross-validation?

Answer: Technique to evaluate model performance.

4. What is feature engineering?

Answer: Creating new features from existing data.

5. Explain the bias-variance tradeoff.

Answer: Balancing model simplicity and accuracy.

6. What is a confusion matrix?

Answer: Table for evaluating classification models.

7. What is PCA?

Answer: Principal Component Analysis for dimensionality reduction.

8. Explain A/B testing.

Answer: Comparing two versions to see which performs better.

9. What is time series analysis?

Answer: Analyzing data points collected over time.

10. What is data normalization?

Answer: Scaling data to a standard range.
