Here are common interview questions on Data Science.

Interview Questions

1. What is the difference between supervised and unsupervised learning?

Answer: Supervised uses labeled data, unsupervised does not.

2. Explain overfitting and underfitting.

Answer: Overfitting fits training data too well, underfitting too poorly.

3. What is cross-validation?

Answer: Technique to evaluate model performance.

4. What is feature engineering?

Answer: Creating new features from existing data.

5. Explain the bias-variance tradeoff.

Answer: Balancing model simplicity and accuracy.

6. What is a confusion matrix?

Answer: Table for evaluating classification models.

7. What is PCA?

Answer: Principal Component Analysis for dimensionality reduction.

8. Explain A/B testing.

Answer: Comparing two versions to see which performs better.

9. What is time series analysis?

Answer: Analyzing data points collected over time.

10. What is data normalization?

Answer: Scaling data to a standard range.
