Data Science MCQs

Test your knowledge with these multiple-choice questions on Data Science.

1. What is the primary goal of data science?

a) To store data

b) To extract insights from data

c) To delete data

Answer: b) To extract insights from data

2. Which library is commonly used for data manipulation in Python?

a) NumPy

b) Pandas

c) Matplotlib

Answer: b) Pandas

3. What does SQL stand for?

a) Structured Query Language

b) Simple Query Language

c) Standard Query Language

Answer: a) Structured Query Language

4. Which technique is used for data visualization?

a) Regression

b) Clustering

c) Plotting

Answer: c) Plotting

5. What is a common machine learning algorithm for classification?

a) Linear Regression

b) Decision Tree

c) K-Means

Answer: b) Decision Tree
