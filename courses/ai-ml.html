How to use: Study each topic, complete the suggested lab, and move to the next topic when comfortable.

1. Setup & Python Basics

  • Install Python, Jupyter, and Git
  • Python basics: variables, lists, dicts, functions
  • Working with notebooks and running simple scripts
  • Lab: "Hello, AI" notebook + simple data printouts

2. Intro to AI & ML Concepts

  • What is AI vs ML vs DL; real-world examples
  • Types: supervised, unsupervised, reinforcement

  • Lab: EDA on a small dataset

4. Basic Math for ML (Intuitive)

  • Mean/variance, probability basics

  • Feature scaling and simple feature engineering
  • Lab: tune a model and compare metrics

7. Intro to Neural Networks

  • Neurons, layers, activations, loss
  • Backpropagation (intuitive view)
  • Using Keras/PyTorch for small nets
  • Lab: MNIST-like classifier with a tiny NN

8. Deep Learning Practices

  • Convolutional nets (basic idea) and transfer learning
  • Data augmentation and regularization
  • Debugging and improving models
  • Lab: fine-tune a pretrained image model on a small set

9. Natural Language Processing (Intro)

  • Tokenization, bag-of-words, embeddings
  • RNN/LSTM basics and transformer overview

  • Lab: sentiment classifier using pretrained embeddings

10. Unsupervised Learning & Dimensionality Reduction

  • Clustering (k-means), PCA overview

11. Deployment Basics & MLOps Intro

  • Save/load models, simple model serving (script/Flask)

  • Ethics, fairness, and privacy checklist
  • Lab: save a model and create a minimal prediction endpoint

12. Capstone Project

  • Collect/clean data ‚Üí train ‚Üí evaluate ‚Üí deploy
  • Deliverables: code repo, notebook, short demo
  • Lab: complete and present capstone
