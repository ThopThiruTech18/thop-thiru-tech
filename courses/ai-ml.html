AI & Machine Learning MCQs

Test your knowledge with these multiple-choice questions on AI & Machine Learning.

1. What is supervised learning?

a) Learning without labels

b) Learning with labeled data

c) Learning from images only

Answer: b) Learning with labeled data

2. Which algorithm is used for regression?

a) K-Means

b) Linear Regression

c) KNN

Answer: b) Linear Regression

3. What is a neural network?

a) A type of database

b) A model inspired by the human brain

c) A programming language

Answer: b) A model inspired by the human brain

4. Which library is popular for machine learning in Python?

a) TensorFlow

b) Django

c) Flask

Answer: a) TensorFlow

5. What is overfitting?

a) Model performs well on training data but poorly on new data

b) Model performs poorly on training data

c) Model has too few parameters

Answer: a) Model performs well on training data but poorly on new data

6. What is unsupervised learning?

a) Learning with labeled data

b) Learning without labeled data

c) Learning from images only

Answer: b) Learning without labeled data

7. Which is a clustering algorithm?

a) Linear Regression

b) K-Means

c) Decision Tree

Answer: b) K-Means

8. What is reinforcement learning?

a) Learning from rewards and punishments

b) Learning from labeled data

c) Learning from clusters

Answer: a) Learning from rewards and punishments

9. What is a convolutional neural network (CNN) used for?

a) Text processing

b) Image recognition

c) Time series forecasting

Answer: b) Image recognition

10. What is bias in machine learning?

a) A type of algorithm

b) Systematic error from incorrect assumptions

c) Random error in data

Answer: b) Systematic error from incorrect assumptions
