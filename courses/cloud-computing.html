Test your knowledge with these multiple-choice questions on Cloud Computing.

1. What is cloud computing?

a) Storing data on local servers

b) Delivering computing services over the internet

c) Using physical hardware only

Answer: b) Delivering computing services over the internet

2. Which is a major cloud service provider?

a) AWS

b) Microsoft Office

c) Adobe Photoshop

Answer: a) AWS

3. What does IaaS stand for?

a) Infrastructure as a Service

b) Internet as a Service

c) Information as a Service

Answer: a) Infrastructure as a Service

4. What is scalability in cloud computing?

a) Ability to handle increased load

b) Reducing costs

c) Storing data locally

Answer: a) Ability to handle increased load

5. Which service provides virtual machines?

a) PaaS

b) SaaS

c) IaaS

Answer: c) IaaS

6. What does PaaS stand for?

a) Platform as a Service

b) Personal as a Service

c) Process as a Service

Answer: a) Platform as a Service

7. Which is an example of SaaS?

a) Google Drive

b) AWS EC2

c) Heroku

Answer: a) Google Drive

8. What is hybrid cloud?

a) Only public cloud

b) Combination of public and private clouds

c) Only private cloud

Answer: b) Combination of public and private clouds

9. What is serverless computing?

a) No servers needed

b) Running code without managing servers

c) Using physical servers

Answer: b) Running code without managing servers

10. Which AWS service is for storage?

a) S3

b) EC2

c) Lambda

Answer: a) S3

Interview Questions

1. What are the benefits of cloud computing?

Answer: Scalability, cost-efficiency, accessibility.

2. Explain IaaS, PaaS, SaaS.

Answer: Infrastructure, Platform, Software as a Service.

3. What is auto-scaling?

Answer: Automatically adjusting resources based on demand.

4. What is a load balancer?

Answer: Distributes traffic across servers.

5. Explain cloud migration.

Answer: Moving applications and data to the cloud.

6. What is multi-cloud strategy?

Answer: Using multiple cloud providers.

7. What is containerization?

Answer: Packaging applications with dependencies, e.g., Docker.

8. Explain microservices architecture.

Answer: Breaking applications into small, independent services.

9. What is DevOps?

Answer: Practices combining development and operations.

10. What is cloud security?

Answer: Protecting data and applications in the cloud.
