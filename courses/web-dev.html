Web Development MCQs

Test your knowledge with these multiple-choice questions on Web Development.

1. What does HTML stand for?

a) Hyper Text Markup Language

b) High Tech Modern Language

c) Home Tool Markup Language

Answer: a) Hyper Text Markup Language

2. Which CSS property is used to change the text color?

a) font-color

b) text-color

c) color

Answer: c) color

3. What is the purpose of JavaScript?

a) To style web pages

b) To add interactivity to web pages

c) To structure web content

Answer: b) To add interactivity to web pages

4. Which tag is used to create a hyperlink in HTML?

a) <link>

b) <a>

c) <href>

Answer: b) <a>

5. What does CSS stand for?

a) Computer Style Sheets

b) Cascading Style Sheets

c) Creative Style Sheets

Answer: b) Cascading Style Sheets

6. What is the difference between id and class in CSS?

a) id is unique, class can be shared

b) class is unique, id can be shared

c) No difference

Answer: a) id is unique, class can be shared

7. Which JavaScript method is used to select an element by ID?

a) getElementById

b) querySelector

c) getElementsByClassName

Answer: a) getElementById

8. What does DOM stand for?

a) Document Object Model

b) Data Object Model

c) Dynamic Object Model

Answer: a) Document Object Model

9. Which HTML element is used for the largest heading?

a) <h1>

b) <h6>

c) <p>

Answer: a) <h1>

10. What is Bootstrap?

a) A JavaScript library

b) A CSS framework

c) A database

Answer: b) A CSS framework
