What is an Array?

An array is a collection of similar type values stored in continuous memory.

int[] numbers = {10, 20, 30, 40};
  • üìå All values must be of the same data type
  • üìå Array size is fixed
  • üìå Index starts from 0

Why Arrays?

  • ‚úî Store multiple values using one variable
  • ‚úî Easy access using index
  • ‚úî Improves performance

Common Array Programs (Interview ‚≠ê‚≠ê‚≠ê)

Find Sum of Elements

int[] a = {10, 20, 30};
int sum = 0;

for (int n : a) {
    sum += n;
}

System.out.println(sum);

Find Largest Element

int[] a = {10, 5, 30, 20};
int max = a[0];

for (int i = 1; i < a.length; i++) {
    if (a[i] > max) {
        max = a[i];
    }
}

System.out.println(max);

Reverse an Array

int[] a = {1, 2, 3};
for (int i = a.length - 1; i >= 0; i--) {
    System.out.println(a[i]);
}

Search an Element

int[] a = {10, 20, 30};
int key = 20;
boolean found = false;

for (int n : a) {
    if (n == key) {
        found = true;
        break;
    }
}

System.out.println(found);
