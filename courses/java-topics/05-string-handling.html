Java Strings (Zero → Hero)

Zero Level – Basics

What is a String?

A String is a sequence (group) of characters.

String name = "Thiru";

In Java:

  • String is a class (not a primitive)
  • Strings are immutable (cannot be changed)

Why String is Special in Java?

  • Used everywhere (names, messages, JSON, URLs, files)
  • Stored in String Constant Pool (SCP)
  • Immutable → Secure + Thread-safe
  • Saves memory by reusing objects

How to Create a String

  1. Using String Literal (MOST COMMON) 
    String s1 = "Java";
String s2 = "Java";

    Note: Both s1 and s2 point to the same object in SCP.

    • ✔ Memory efficient
    • ✔ Faster
  2. Using new Keyword 
    String s3 = new String("Java");

    Note: Creates a new object in Heap memory.

    • ❌ Not memory efficient
    • ✔ Used when a separate object is required

String Memory (IMPORTANT ⭐)

String s1 = "Java";
String s2 = "Java";
String s3 = new String("Java");
Variable Memory Location
s1 SCP
s2 SCP (same reference as s1)
s3 Heap (new object)

String Immutability (Core Concept)

Definition: Once a String object is created, it cannot be changed.

String s = "Java";
s.concat(" World");

System.out.println(s);

Output: Java

Explanation:

  • "Java World" is created, but not assigned
  • Original string remains unchanged

Why String is Immutable?

  • ✔ Security (passwords, URLs)
  • ✔ Thread safety
  • ✔ Memory efficiency

Mutable Strings (When modification is needed)

StringBuilder

StringBuilder sb = new StringBuilder("Java");
sb.append(" World");

System.out.println(sb);
  • ✔ Mutable
  • ✔ Fast
  • ❌ Not Thread-Safe

StringBuffer

StringBuffer sb = new StringBuffer("Java");
sb.append(" World");

System.out.println(sb);
  • ✔ Mutable
  • ✔ Thread-Safe
  • ❌ Slower than StringBuilder

Comparison (Interview ⭐)

Feature String StringBuilder StringBuffer
Mutable ❌ ✅ ✅
Thread-Safe ✅ ❌ ✅
Performance Slow Fast Medium

Common String Methods (Very Important)

  • length() 
    String s = "Java";
System.out.println(s.length()); // 4
  • charAt() 
    System.out.println(s.charAt(0)); // J
  • equals() vs == 
    String a = "Java";
String b = "Java";

System.out.println(a == b);      // true (reference)
System.out.println(a.equals(b)); // true (content)

    Note: Always use equals() for content comparison.

  • equalsIgnoreCase() 
    "java".equalsIgnoreCase("JAVA"); // true
  • toUpperCase() / toLowerCase() 
    "Java".toUpperCase(); // JAVA
  • substring() 
    String s = "Programming";
System.out.println(s.substring(0, 6)); // Progra
  • contains() 
    "Java Full Stack".contains("Full"); // true
  • startsWith() / endsWith() 
    "Java".startsWith("J"); // true
  • replace() 
    "Java".replace("a", "o"); // Jovo
  • split() 
    String s = "Java,Spring,Angular";
String[] arr = s.split(",");
  • trim() 
    "  Java  ".trim(); // Java

String Programs (Interview ⭐⭐⭐)

  • Reverse a String 
    String s = "Java";
String rev = "";

for (int i = s.length() - 1; i >= 0; i--) {
    rev += s.charAt(i);
}

System.out.println(rev); // avaJ
  • Palindrome String 
    String s = "madam";
String rev = new StringBuilder(s).reverse().toString();

System.out.println(s.equals(rev)); // true
  • Count Characters 
    String s = "Java";
System.out.println(s.length());
  • Count Words 
    String s = "Java Full Stack";
String[] words = s.split(" ");
System.out.println(words.length);

Advanced (Hero Level)

  • String Interning 
    String s1 = new String("Java").intern();
String s2 = "Java";

    Note: Both point to the same SCP object.

  • StringTokenizer 
    StringTokenizer st = new StringTokenizer("Java Spring Boot");

while (st.hasMoreTokens()) {
    System.out.println(st.nextToken());
}

    Note: Used to break a string into tokens.

  • StringJoiner (Java 8) 
    StringJoiner sj = new StringJoiner(",");
sj.add("Java")
  .add("Spring")
  .add("Microservices");

System.out.println(sj);

Interview One-Line Answers ⭐

  • String → Immutable class
  • equals() → Content comparison
  • == → Reference comparison
  • StringBuilder → Fast & mutable
  • StringBuffer → Thread-safe

Final Summary (Hero Level)

  • ✔ String is immutable
  • ✔ Use StringBuilder for modifications
  • ✔ Use equals(), not ==
  • ✔ Strings are the backbone of Java apps
