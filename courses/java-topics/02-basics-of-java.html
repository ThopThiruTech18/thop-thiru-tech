Basics of Java

1️⃣ Variables

Variables are containers for storing data values. In Java, you must declare a variable with a specific data type.

int age = 25;
String name = "John";
boolean isJavaFun = true;

Explanation: In the above example:

  • int age = 25; declares an integer variable age with a value of 25.
  • String name = "John"; declares a string variable name with the value "John".
  • boolean isJavaFun = true; declares a boolean variable isJavaFun with the value true.

2️⃣ Data Types

Java is a statically-typed language, meaning every variable must have a declared data type.

  • Primitive Types: int, float, double, char, boolean, byte, short, long
  • Non-Primitive Types: String, Arrays, Classes
int number = 10;       // Primitive type
String text = "Hello"; // Non-primitive type

3️⃣ Type Casting

Type casting is used to convert one data type into another.

  • Implicit Casting: Automatically done by Java (e.g., int to double).
  • Explicit Casting: Manually done by the programmer (e.g., double to int).
double myDouble = 9.78;
int myInt = (int) myDouble; // Explicit casting

4️⃣ Operators

Java provides various operators to perform operations on variables and values.

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !
int a = 10, b = 20;
System.out.println(a + b); // Addition
System.out.println(a > b); // Relational
System.out.println(a > 5 && b > 15); // Logical

5️⃣ Keywords & Identifiers

Keywords: Reserved words in Java (e.g., class, public, static).

Identifiers: Names given to variables, methods, classes, etc.

class MyClass {
    int myVariable;
    void myMethod() {
        // Method body
    }
}

6️⃣ Comments

Comments are used to explain code and are ignored by the compiler.

  • Single-line Comment: // This is a comment
  • Multi-line Comment: 
    /*
This is a multi-line comment
*/

7️⃣ Input & Output

Java provides the Scanner class for user input and System.out for output.

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter your name: ");
        String name = scanner.nextLine();
        System.out.println("Hello, " + name);
    }
}
