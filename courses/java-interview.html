Here are 100 common Java interview questions with answers.

1. What is Java?

Answer: Java is a high-level, object-oriented programming language developed by Sun Microsystems (now Oracle).

2. What are the main features of Java?

Answer: Platform independence, object-oriented, secure, robust, multithreaded, etc.

3. What is JVM?

Answer: Java Virtual Machine, which executes Java bytecode.

4. What is JRE?

Answer: Java Runtime Environment, includes JVM and libraries.

5. What is JDK?

Answer: Java Development Kit, includes JRE and development tools.

6. What is the difference between JDK, JRE, and JVM?

Answer: JDK is for development, JRE for running, JVM is the virtual machine.

7. What is a class in Java?

Answer: A blueprint for objects, containing methods and variables.

8. What is an object in Java?

Answer: An instance of a class.

9. What is inheritance?

Answer: Mechanism where one class acquires properties of another.

10. What is polymorphism?

Answer: Ability of an object to take many forms.

11. What is encapsulation?

Answer: Wrapping data and methods into a single unit.

12. What is abstraction?

Answer: Hiding implementation details and showing only functionality.

13. What are access modifiers in Java?

Answer: public, private, protected, default.

14. What is the difference between abstract class and interface?

Answer: Abstract class can have concrete methods, interface cannot (before Java 8).

15. What is a constructor?

Answer: Special method to initialize objects.

16. What is method overloading?

Answer: Same method name with different parameters.

17. What is method overriding?

Answer: Subclass provides specific implementation of superclass method.

18. What is the super keyword?

Answer: Refers to superclass.

19. What is the this keyword?

Answer: Refers to current object.

20. What is a static variable?

Answer: Shared among all instances of the class.

21. What is a static method?

Answer: Can be called without creating an instance.

22. What is final keyword?

Answer: Used to make variables, methods, or classes unchangeable.

23. What is a String in Java?

Answer: Immutable sequence of characters.

24. What is StringBuilder?

Answer: Mutable sequence of characters.

25. What is the difference between == and equals()?

Answer: == checks reference, equals() checks content.

26. What is an ArrayList?

Answer: Resizable array implementation of List interface.

27. What is a HashMap?

Answer: Key-value pair storage.

28. What is an exception?

Answer: An event that occurs during execution disrupting normal flow.

29. What is try-catch?

Answer: Blocks to handle exceptions.

30. What is finally block?

Answer: Executes regardless of exception.

31. What is multithreading?

Answer: Running multiple threads simultaneously.

32. What is synchronization?

Answer: Controlling access to shared resources.

33. What is a thread?

Answer: Lightweight subprocess.

34. What is Runnable interface?

Answer: Interface for creating threads.

35. What is garbage collection?

Answer: Automatic memory management.

36. What is serialization?

Answer: Converting object to byte stream.

37. What is deserialization?

Answer: Converting byte stream back to object.

38. What is a lambda expression?

Answer: Anonymous function for functional programming.

39. What is a stream in Java 8?

Answer: Sequence of elements supporting sequential and parallel operations.

40. What is Optional in Java?

Answer: Container for potentially null values.

41. What is the main method?

Answer: Entry point of Java application.

42. What is a package?

Answer: Group of related classes and interfaces.

43. What is import statement?

Answer: Brings classes from other packages.

44. What is a jar file?

Answer: Java Archive file containing compiled classes.

45. What is JDBC?

Answer: Java Database Connectivity for database access.

46. What is Hibernate?

Answer: ORM framework for Java.

47. What is Spring Framework?

Answer: Framework for enterprise Java applications.

48. What is Maven?

Answer: Build automation tool.

49. What is Gradle?

Answer: Build tool similar to Maven.

50. What is JUnit?

Answer: Testing framework for Java.

51. What is the difference between ArrayList and LinkedList?

Answer: ArrayList is array-based, LinkedList is node-based.

52. What is the difference between HashMap and HashTable?

Answer: HashMap is not synchronized, HashTable is.

53. What is the difference between Iterator and ListIterator?

Answer: ListIterator can traverse backwards.

54. What is the difference between throw and throws?

Answer: throw throws exception, throws declares it.

55. What is checked exception?

Answer: Checked at compile time.

56. What is unchecked exception?

Answer: Checked at runtime.

57. What is NullPointerException?

Answer: Thrown when accessing null object.

58. What is ArrayIndexOutOfBoundsException?

Answer: Thrown when accessing invalid array index.

59. What is ClassNotFoundException?

Answer: Thrown when class not found.

60. What is NoClassDefFoundError?

Answer: Thrown when class definition not found at runtime.

61. What is the difference between process and thread?

Answer: Process is heavyweight, thread is lightweight.

62. What is deadlock?

Answer: Situation where threads wait indefinitely.

63. What is volatile keyword?

Answer: Ensures visibility of changes to variables across threads.

64. What is the difference between wait() and sleep()?

Answer: wait() releases lock, sleep() does not.

65. What is the difference between notify() and notifyAll()?

Answer: notify() wakes one thread, notifyAll() wakes all.

66. What is the difference between String, StringBuffer, and StringBuilder?

Answer: String is immutable, StringBuffer is synchronized mutable, StringBuilder is non-synchronized mutable.

67. What is autoboxing?

Answer: Automatic conversion of primitive to wrapper.

68. What is unboxing?

Answer: Automatic conversion of wrapper to primitive.

69. What is generics?

Answer: Allows type parameters.

70. What is the difference between List and Set?

Answer: List allows duplicates, Set does not.

71. What is the difference between HashSet and TreeSet?

Answer: HashSet unordered, TreeSet ordered.

72. What is the difference between Array and ArrayList?

Answer: Array fixed size, ArrayList resizable.

73. What is the difference between Comparable and Comparator?

Answer: Comparable natural ordering, Comparator custom.

74. What is the difference between fail-fast and fail-safe iterators?

Answer: Fail-fast throws exception on modification, fail-safe does not.

75. What is the difference between HashMap and ConcurrentHashMap?

Answer: ConcurrentHashMap is thread-safe.

76. What is the difference between Vector and ArrayList?

Answer: Vector is synchronized, ArrayList is not.

77. What is the difference between Stack and Queue?

Answer: Stack LIFO, Queue FIFO.

78. What is the difference between LinkedHashMap and HashMap?

Answer: LinkedHashMap maintains insertion order.

79. What is the difference between TreeMap and HashMap?

Answer: TreeMap sorted, HashMap not.

80. What is the difference between EnumSet and HashSet?

Answer: EnumSet for enums, more efficient.

81. What is the difference between FileInputStream and FileReader?

Answer: FileInputStream for bytes, FileReader for characters.

82. What is the difference between BufferedReader and FileReader?

Answer: BufferedReader buffered, more efficient.

83. What is the difference between PrintWriter and FileWriter?

Answer: PrintWriter has print methods.

84. What is the difference between Serializable and Externalizable?

Answer: Externalizable allows custom serialization.

85. What is the difference between transient and volatile?

Answer: transient excludes from serialization, volatile for threads.

86. What is the difference between assert and if?

Answer: assert for debugging, can be disabled.

87. What is the difference between break and continue?

Answer: break exits loop, continue skips iteration.

88. What is the difference between do-while and while?

Answer: do-while executes at least once.

89. What is the difference between for and while?

Answer: for known iterations, while condition-based.

90. What is the difference between switch and if-else?

Answer: switch for multiple constants.

91. What is the difference between int and Integer?

Answer: int primitive, Integer wrapper.

92. What is the difference between float and double?

Answer: double has more precision.

93. What is the difference between char and String?

Answer: char single character, String sequence.

94. What is the difference between byte and short?

Answer: byte 8-bit, short 16-bit.

95. What is the difference between long and int?

Answer: long 64-bit, int 32-bit.

96. What is the difference between boolean and Boolean?

Answer: boolean primitive, Boolean wrapper.

97. What is the difference between void and return?

Answer: void no return, return specifies value.

98. What is the difference between static and instance?

Answer: static class-level, instance object-level.

99. What is the difference between public and private?

Answer: public accessible everywhere, private only in class.

100. What is the difference between protected and default?

Answer: protected subclass access, default package access.
