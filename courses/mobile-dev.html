MCQs


1. Which language is used for Android development?

a) Swift

b) Kotlin

c) Objective-C

Answer: b) Kotlin

2. What is Flutter?

a) A database

b) A UI toolkit for cross-platform apps

c) A server framework

Answer: b) A UI toolkit for cross-platform apps

3. Which is used for iOS development?

a) Java

b) Swift

c) Python

Answer: b) Swift

4. What is APK in Android?

a) Application Package

b) Android Package Kit

c) App Programming Key

Answer: b) Android Package Kit

5. Which framework is for React Native?

a) Angular

b) Vue.js

c) React

Answer: c) React

Interview Questions


1. Difference between native and hybrid apps?

Answer: Native apps are platform-specific, hybrid use web tech.

2. Explain Android app lifecycle.

Answer: onCreate, onStart, onResume, onPause, onStop.

3. What is a fragment?

Answer: Reusable UI component.

4. What is SwiftUI?

Answer: Declarative UI framework for iOS.

5. Explain push notifications.

Answer: Messages sent to users from server.
