Test your knowledge with these multiple-choice questions on Mobile Development.

1. Which language is used for Android development?

a) Swift

b) Kotlin

c) Objective-C

Answer: b) Kotlin

2. What is Flutter?

a) A database

b) A UI toolkit for cross-platform apps

c) A server framework

Answer: b) A UI toolkit for cross-platform apps

3. Which is used for iOS development?

a) Java

b) Swift

c) Python

Answer: b) Swift

4. What is APK in Android?

a) Application Package

b) Android Package Kit

c) App Programming Key

Answer: b) Android Package Kit

5. Which framework is for React Native?

a) Angular

b) Vue.js

c) React

Answer: c) React

6. What is Xcode used for?

a) Android development

b) iOS development

c) Web development

Answer: b) iOS development

7. What is an emulator in mobile development?

a) A physical device

b) Software that mimics a mobile device

c) A programming language

Answer: b) Software that mimics a mobile device

8. Which is a popular Android IDE?

a) Xcode

b) Android Studio

c) Visual Studio Code

Answer: b) Android Studio

9. What is a widget in mobile apps?

a) A small interactive component on the home screen

b) A full application

c) A database table

Answer: a) A small interactive component on the home screen

10. What is app store optimization (ASO)?

a) Optimizing app performance

b) Optimizing app visibility in app stores

c) Optimizing app design

Answer: b) Optimizing app visibility in app stores

Interview Questions

1. What is the difference between native and hybrid apps?

Answer: Native for specific platform, hybrid use web tech in native shell.

2. Explain the app lifecycle in Android.

Answer: onCreate, onStart, onResume, etc.

3. What is a fragment in Android?

Answer: Reusable UI component.

4. What is SwiftUI?

Answer: Declarative UI framework for iOS.

5. Explain push notifications.

Answer: Messages sent to app users.

6. What is app signing?

Answer: Process to verify app authenticity.

7. What is a mobile API?

Answer: Interface for app to communicate with servers.

8. Explain in-app purchases.

Answer: Buying content within the app.

9. What is mobile testing?

Answer: Testing apps on devices and emulators.

10. What is app monetization?

Answer: Ways to earn money from apps, like ads or subscriptions.
