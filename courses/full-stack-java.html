Here are common interview questions on Full Stack Java.

Interview Questions

1. What is full stack development?

Answer: Development of both frontend and backend of web applications.

2. What is Spring Boot?

Answer: A framework to simplify Spring application development.

3. Explain dependency injection.

Answer: A technique where an object receives its dependencies from an external source.

4. What is Hibernate?

Answer: An ORM framework for mapping Java objects to database tables.

5. What is REST API?

Answer: An API that uses HTTP requests to GET, PUT, POST, DELETE data.

6. How do you handle exceptions in Spring?

Answer: Using @ExceptionHandler or global exception handling.

7. What is JPA?

Answer: Java Persistence API, a specification for ORM.

8. Difference between @RequestMapping and @GetMapping?

Answer: @RequestMapping is general, @GetMapping is for GET requests.

9. What is Maven used for?

Answer: Dependency management and build automation.

10. How to secure a REST API in Spring?

Answer: Use Spring Security with JWT or OAuth.
