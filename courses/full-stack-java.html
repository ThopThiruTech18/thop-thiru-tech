Here are common interview questions on Full Stack Java.

1. What is full stack development?

Development of both frontend and backend of web applications.

2. What is Spring Boot?

Framework to simplify Spring application development.

3. Explain dependency injection.

Object receives dependencies from an external source.

4. What is Hibernate?

ORM framework for mapping Java objects to database tables.

5. What is REST API?

Uses HTTP methods to perform CRUD operations.

6. How do you handle exceptions in Spring?

Using @ExceptionHandler or @ControllerAdvice.

7. What is JPA?

Java Persistence API specification for ORM.

8. Difference between @RequestMapping and @GetMapping?

@RequestMapping is generic, @GetMapping is specific to GET.

9. What is Maven used for?

Dependency management and build automation.

10. How to secure a REST API in Spring?

Using Spring Security with JWT or OAuth2.
